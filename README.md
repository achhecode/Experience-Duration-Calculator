# Experience Duration Calculator

A simple Python project to calculate total work experience (years and months)
from a JSON file containing multiple experience entries.

---

## Features
- Reads experience data from JSON
- Uses clean OOP design
- Calculates total experience in months, years
- Easy to extend (Present date, overlap handling, validation)
